

Copy images and set macOS creation time from EXIF metadata

Tested to work on macOS, open to contributions for linux and windows versions

```
options:
  -h, --help            show this help message and exit
  -i, --input INPUT_DIR
                        Input directory containing images (default: ./input1)
  -o, --output OUTPUT_DIR
                        Output directory to copy files to (default: ./output1)
```
